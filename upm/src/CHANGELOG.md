# Changelog

All notable changes to `com.noodlehammer.tools` will be documented in this file.

## [0.1.0] - 2026-06-14

### Added

- `Noodle Hammer Hierarchy` with hierarchy visuals, styling, and settings support.
- `Noodle Hammer Animator` with inspector playback helpers and Avatar T-pose utilities.
- `Noodle Hammer Transform` with quick transform reset, copy, and paste actions.
- Package documentation covering the mirrored UPM structure and installation context.
